## Introduction

PacMan is a classic arcade game where the player navigates PacMan through a maze, eating dots while avoiding ghosts. The objective is to clear the maze of all dots while maximizing the score.

## Contributing

We welcome contributions from the community! If you would like to contribute to the PacMan game, please follow these steps:

1. Fork the [PacMan Game repository](https://github.com/CNAM-A1-Pacman/Pacman).
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with clear messages.
4. Push your changes to your forked repository.
5. Open a pull request to the main repository.
Please ensure that your code adheres to the project's coding standards and includes appropriate tests. We look forward to your contributions!
For any questions or further assistance, feel free to open an issue or contact the project maintainers.
Happy gaming and coding!
